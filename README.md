# ios_starter_kit

iOS Swift Project Boilerplate.

## Requirements

* Swift 2.2+
* XCode 7+

## Install Dependencies

### swift lint

```
brew install swiftlint
```

### Xcode Plugins

Recommend Alcatraz.

* Swimat
    * 2 Space Indent
    * Format when Save
    * Format when Build
* KZLinkedConsole
* XcodeColors
